Block Elements

Block elements start each element on a new line and take up the entire width of the page. Examples are: div,h1,p,ul,ol,section,article,footer, and header.

Inline Elements

Inline elements do not start on a new line and only take up as much width as they need to. Example are: a,img,and span

Inline-Block

This is used as a display property in the CSS file. It allows a block element to act like an Inline element. This only works if you add a width to the elements.
